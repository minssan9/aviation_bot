# Aviation Bot Admin Frontend

Vue 3 + TypeScript + Vite + Quasar admin interface for managing the Aviation Bot backend.

## Development

```bash
npm install
npm run dev
```

The frontend dev server will run on `http://localhost:5173` and proxy API requests to `http://localhost:3010`.

## Build

```bash
npm run build
```

Build output will be in the `dist/` directory, which is served by the backend in production.
